# 1. Descargar Ventoy de la página Oficial

[*Página Oficial*](https://www.ventoy.net/en/download.html)

# 2. Instalar Ventoy

- Para usuarios Windows, simplemente descargar el **archivo.exe** correspondiente y ejecutar.

- Para usuarios Linux, descargar el **archivo.linux.tar.gz** . Una vez lo hayais descomprimido, entrar dentro de la carpeta, y ejecutar el archivo con el nombre ***VentoyGUI.x86_64***.

![Ventoy 1](ventoy1.png)

# 3. Instalar Ventoy en el USB.

Una vez tengamos acceso a esa interfaz, simplemente conectamos el dispositivo USB que vamos a utilizar para poner todas nuestras imágenes.

![Ventoy 2](ventoy2.png)

Le damos a instalar y aceptamos que formatee el dispositivo USB (**MUY IMPORTANTE, ESTO BORRARA TODO LO QUE HAYA EN EL USB**).

Una vez instalado Ventoy, aparecerá en la parte de **Ventoy in Device** la versión que se ha instalado y tendremos una carpeta como un USB normal y corriente al que le arrastramos imágenes ISO para poder almacenar ahí las imágenes a ejecutar.

Una vez se han introducido varias imágenes. Apagar el PC y probar usar este USB como si fuese uno booteable seleccionandolo en la opción de arranque. Aparecerá un menú asi.

![Ventoy 3](ventoy3.png)
